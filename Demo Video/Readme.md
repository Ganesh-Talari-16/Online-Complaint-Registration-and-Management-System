# Online Complaint Registration and Management System

## 📌 Project Overview
This project is a MERN stack application designed to streamline the process of registering and managing complaints online. It provides a user‑friendly interface for citizens to submit complaints and for administrators to track, manage, and resolve them efficiently.

## 🚀 Features
- User registration and login
- Complaint submission with details
- Admin dashboard for complaint management
- Status tracking and updates
- Secure authentication and data handling

## 🛠️ Tech Stack
- **Frontend:** React.js, HTML, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Version Control:** Git & GitHub
