# Online Complaint Registration and Management System

## 📌 Project Overview
A MERN stack application designed to streamline complaint registration and management. It enables users to submit complaints online and provides administrators with tools to track, manage, and resolve them efficiently.

## 🚀 Features
- User registration and login  
- Complaint submission with details  
- Admin dashboard for complaint management  
- Status tracking and updates  
- Secure authentication and data handling  

## 🛠️ Tech Stack
- **Frontend:** React.js, HTML, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Version Control:** Git & GitHub
